# DomnaModel: Пофакторний аналіз доменної плавки

Вебдодаток для автоматизації розрахунків зміни питомої витрати коксу та продуктивності доменної печі згідно із **Загальногалузевою методикою пофакторного аналізу доменної плавки**.

## Призначення
Інструмент призначений для інженерно-технічного персоналу металургійних підприємств та дослідників. Дозволяє оперативно оцінити вплив технологічних факторів на основні показники плавки при зміні вхідних умов.

## Ключові особливості
* **Дві базові технології:** підтримка розрахунків для режимів із використанням природного газу (ПГ) та пиловугільного палива (ПВП/ПУТ).
* **Актуальність даних:** враховано оновлення нормативів для показників міцності коксу (CSR/M25), заміни паливом та використання ПВП.
* **Інтерактивність:** перерахунок результатів у реальному часі при виборі факторів та введенні значень.
* **Адаптивність:** інтерфейс оптимізований для використання на різних типах пристроїв.

## Методична база
Розрахункові коефіцієнти відповідають **загальногалузевій методиці пофакторного аналізу доменної плавки** (вплив технологічних факторів на питому витрату коксу та продуктивність доменної печі).
